# Exemplo simples

Exemplo simples de provisionamento de recursos do shared-vpc-service.

## Uso

Para aplicar este exemplo, execute:

```console
$ terraform init
$ terraform plan
$ terraform apply
```

Quando conveniente, remova os recursos criados:

```console
$ terraform destroy
```
